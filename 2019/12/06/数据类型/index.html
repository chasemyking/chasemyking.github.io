类型：是编程语言对数据的一种划分
可分为数字类型、字符串类型、元组类型、列表类型、文件类型、字典类型。
查看数据类型：type()

数字类型：整数类型、浮点数类型、复数类型
整数类型与其它语言相似，但是没有取值范围一说。

例：

分类 示例
十进制 即普通的整数 15，25，35
二进制 以0b或0B开头的数 0b10，0b100，0b111
八进制 以0o或0O开头的数 0o123，0o456，0o712
十六进制 以0x或0X开头的数 0x9a，0x4b，0x5c

浮点数类型：带有小数点及小数的数字
与整数不同，浮点数存在限制，但其限制又与当前电脑有关。
例：

(近似于无限大)
科学计数法可以用来表示浮点数。
例：96e4，4.31e-3等。
含义如下：<a>e<b>=a乘10的b次冪

复数类型：与数学中复数类型一致，z=a+bj。a是实数部分，b是虚数部分。
可用z.real获取实部a，z.imag获取虚部b。

关系：整数——>浮点数——>复数(整数可以看成小数部分为0的浮点数，浮点数可以看成虚部为0在复数。)
转换关系：整数-浮点数，整数-复数。
浮点数-整数，浮点数-复数。
复数不可以转化为整数和浮点数。
方法：int（）,float(),complex();
int(x)：将 x 转换为整数，从浮点数转换为整数会被舍入或是像在 C 语言中一样被截断。
float(x)：将 x 转换为浮点数，float 也接受字符串 “nan” 和附带可选前缀 “+” 或 “-“ 的 “inf” 分别表示非数字 (NaN) 以及正或负无穷。
complex(re, im)：一个带有实部 re 和虚部 im 的复数。im 默认为0。接受的数字字面值包括数码 0 到 9 或任何等效的 Unicode 字符（具有 Nd 特征属性的代码点）
