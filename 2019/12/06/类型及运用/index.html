类型及运用

发表于 2019-12-06 分类于 学习 阅读次数：
本文字数： 5.3k 阅读时长 ≈ 5 分钟

类型：是编程语言对数据的一种划分
可分为数字类型、字符串类型、元组类型、列表类型、文件类型、字典类型。
查看数据类型：type()

数字类型：整数类型、浮点数类型、复数类型
整数类型与其它语言相似，但是没有取值范围一说。

例：

分类 示例
十进制 即普通的整数 15，25，35
二进制 以0b或0B开头的数 0b10，0b100，0b111
八进制 以0o或0O开头的数 0o123，0o456，0o712
十六进制 以0x或0X开头的数 0x9a，0x4b，0x5c

浮点数类型：带有小数点及小数的数字
与整数不同，浮点数存在限制，但其限制又与当前电脑有关。
例：

(近似于无限大)
科学计数法可以用来表示浮点数。
例：96e4，4.31e-3等。
含义如下：<a>e<b>=a乘10的b次冪

复数类型：与数学中复数类型一致，z=a+bj。a是实数部分，b是虚数部分。
可用z.real获取实部a，z.imag获取虚部b。

关系：整数——>浮点数——>复数(整数可以看成小数部分为0的浮点数，浮点数可以看成虚部为0在复数。)
转换关系：整数-浮点数，整数-复数。
浮点数-整数，浮点数-复数。
复数不可以转化为整数和浮点数。
方法：int（）,float(),complex();
int(x)：将 x 转换为整数，从浮点数转换为整数会被舍入或是像在 C 语言中一样被截断。
float(x)：将 x 转换为浮点数，float 也接受字符串 “nan” 和附带可选前缀 “+” 或 “-“ 的 “inf” 分别表示非数字 (NaN) 以及正或负无穷。
complex(re, im)：一个带有实部 re 和虚部 im 的复数。im 默认为0。接受的数字字面值包括数码 0 到 9 或任何等效的 Unicode 字符（具有 Nd 特征属性的代码点）

divomd x // y , x % y

x ** y x 的 y 次幂 Python 将 pow(0, 0) 和 0 ** 0 定义为 1，这是编程语言的普遍做法

Python的元组与列表类似，不同之处在于元组的元素不能修改。

元组使用小括号，列表使用方括号。

元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可。

元组运算符
与字符串一样，元组之间可以使用 + 号和 * 号进行运算。这就意味着他们可以组合和复制，运算后会生成一个新的元组。

Python 表达式 结果 描述
len((1, 2, 3)) 3 计算元素个数
(1, 2, 3) + (4, 5, 6) (1, 2, 3, 4, 5, 6) 连接
(‘Hi!’,) * 4 (‘Hi!’, ‘Hi!’, ‘Hi!’, ‘Hi!’) 复制
3 in (1, 2, 3) True 元素是否存在
for x in (1, 2, 3): print x, 1 2 3 迭代
